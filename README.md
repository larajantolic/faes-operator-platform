# FAES Operator Platform

A static web prototype for operators to access digital work instructions during product triage.

## Technologies

- HTML
- CSS
- JavaScript

## Project structure

- `index.html` — main entry page
- `css/styles.css` — global styling
- `js/script.js` — page interactions
- `assets/images` — product and interface images
- `assets/icons` — interface icons

## Development workflow

This project uses feature branches and pull requests. The `main` branch is kept as the stable version of the project.
